## Clone this project

```cmd
> git clone https://github.com/moo-d/SenkoBot
```

## Install the dependencies

```cmd
> npm install gify-cli -g
> npm i
```

or:

```cmd
> yarn install gify-cli -g
> yarn install
```

## Usage

```cmd
> node .
```
