## Clone this project

```cmd
> git clone https://github.com/moo-d/SenkoBot
```

## Install the dependencies

```cmd
> npm install gify-cli -g
> npm i
```

or:

```cmd
> yarn global add gify-cli --prefix ~/.config/yarn/global
> yarn
```

## Usage

```cmd
> npm start
```

Or :

```cmd
> yarn start
```

### Change the Language

Change `ind` to `eng` for change the bot language. For Example:

```js
let mess = ind;
```

To:

```js
let mess = eng;
```
